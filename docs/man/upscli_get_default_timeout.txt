UPSCLI_GET_DEFAULT_TIMEOUT(3)
=============================

NAME
----

upscli_get_default_timeout - Return the upsclient module default timeout.

SYNOPSIS
--------

------
	#include <upsclient.h>

	int upscli_get_default_timeout(struct timeval *ptv);
------

DESCRIPTION
-----------

The *upscli_get_default_timeout()* function takes the pointer 'ptv' to a
'struct timeval' and returns the default network timeout last assigned
with *upscli_set_default_timeout()*.
